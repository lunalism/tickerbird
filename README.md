## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.17+
- npm / yarn / pnpm

### Installation

```bash
# 1. Clone
git clone https://github.com/lunalism/alphaboard.git
cd alphaboard

# 2. Install dependencies
npm install

# 3. Setup environment variables
cp .env.local.example .env.local

# 4. Run dev server
npm run dev
```
